# CamperCraft 3D

Веб-приложение для планирования компоновки жилого модуля (кемпера). Позволяет размещать основные модули (кровать, кухня, шкаф), настраивать их размеры, положение, цвет и ориентацию в пространстве.

## Основные возможности (Этап 1)
- **Визуализация:** 3D-изометрия и вид сверху (Top view).
- **Редактирование:** Изменение размеров, координат и поворота модулей.
- **Интерактивность:** Перетаскивание модулей мышью (drag-and-drop), изменение размеров за угловые/боковые маркеры.
- **Валидация:** Автоматическое отслеживание пересечений модулей и выхода за габариты кузова.
- **Сохранение/Загрузка:** Поддержка локального хранилища (LocalStorage) и импорт/экспорт файлов `.camper.json`.
- **Экспорт:** Сохранение эскизов в PNG (стандартный набор, текущий вид, изометрия).
